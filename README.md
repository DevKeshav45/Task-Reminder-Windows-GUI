 
# Task Reminder Desktop Application



## Overview
This Task Reminder Desktop Application is a user-friendly tool that helps individuals efficiently manage and stay updated on their tasks by providing timely desktop notifications.

## Project Details
Language: Python

GUI Toolkit: Tkinter

Notification Library: Plyer

## Project Status (May 2022)
The application is functional, allowing users to create tasks and receive desktop notifications for reminders.
The core features, including User Registration and Login, Task Creation, and Task Reminder Notifications, have been successfully implemented.
Features and Functionality
User Registration and Login:

Users can register and create accounts to personalize their task management experience.
A secure login system ensures the privacy and security of user data.
Task Creation:

Users can create tasks by specifying task details such as title, description, due date, and priority.
The application allows users to set reminders for their tasks.
Task Reminder Notification:

The application provides timely desktop notifications to remind users of their upcoming tasks.
Notifications include essential task details to enhance productivity and time management.
